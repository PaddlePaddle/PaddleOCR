## 2. 安装其他依赖

### 2.1  VQA所需依赖
* paddleocr

```bash
pip3 install paddleocr
```

* PaddleNLP
```bash
git clone https://github.com/PaddlePaddle/PaddleNLP -b develop
cd PaddleNLP
pip3 install -e .
```
