这张图片来自麦肯锡公司（McKinsey & Company）的《全球银行业年度回顾2024：实现逃逸速度》（Global Banking Annual Review 2024: Attaining escape velocity），图中包含了一些关于全球金融中介的数据和图表。

1. **标题和引言**：
   - 标题为“The industry”（行业）。
   - 引用了未知来源的一句话：“Everything is going to be fine in the end. If it’s not fine, it’s not the end.”（一切都会好起来的。如果还没好，那说明还没到结局。）

2. **正文内容**：
   - 文章提到，尽管这句话带有幽默感，但最近银行业的情况确实很好。过去两年是自大萧条以来银行业表现最好的时期。
   - 全球银行在2023年创造了7万亿美元的收入和11万亿美元的净收入。
   - 银行资本充足率（ROTE）达到11.7%，流动性达到77.2%，均较2022年有所改善。
   - 银行业在2023年比其他任何行业都创造了更多的总收入。

3. **图表内容**：
   - **Exhibit 1**：2023年全球金融系统中介了410万亿美元的资产，创造了约7万亿美元的收入。
     - 图表展示了不同类型的资金来源和用途，包括银行负债、企业存款、个人存款、主权财富基金、零售资产管理（AUM）、私人资本、保险和养老金资产管理（AUM）、其他机构、数字资产等。
   - **Exhibit 2**：2023年全球金融中介的年度收入来源，按类型划分。
     - 零售银行业务占33%，支付业务占16%，企业和商业银行占28%，投资银行业务占5%，财富和资产管理占14%，市场基础设施占2%，其他占3%。

4. **数据来源**：
   - 数据来源为麦肯锡（McKinsey）和Paradigm。

这张图片提供了关于全球银行业在2023年的表现和收入来源的详细数据，展示了金融中介行业的现状和趋势。
