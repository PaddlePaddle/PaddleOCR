"这张图片展示了一段关于领导力和适应性的文本。文本讨论了在压力或不确定性下，团队和个人常常依赖他们已知的方法，即使这种方法可能不适用于当前的情况。文中引用了NBA球员Stephen Curry的例子，说明在季后赛中，尽管在第一节结束时错失了一个重要的三分球，他仍然继续在第二节投三分球，因为他相信自己能够投进。然而，如果NBA限制季后赛中三分球的数量，库里可能需要重新思考他的策略，这就是适应性。\n\n文本还引用了麦肯锡健康研究所（McKinsey Health Institute）的一项研究，该研究表明，那些在韧性和适应性方面得分高的人，比他们的同事表现出更好的健康和更高的工作投入度。同时，那些在这两个维度上都得分高的人，比他们的同事更有可能报告高工作投入度和创新行为。\n\n为了实现韧性和适应性，领导者和员工需要具备一系列技能。韧性包括将变化视为挑战或机会的能力，调节思想和情绪，从过去的经验中吸取教训，并执行变革。适应性则需要以开放、学习的心态面对不确定性，灵活和创造性地思考问题。当领导者和员工具备这些技能时，他们可以更好地判断何时需要韧性，何时需要适应性，以及如何将两者整合到他们的战略、运营和决策中。\n\n文本的底部还提到了麦肯锡健康研究所进行的一项跨国调查，该调查涉及30个国家，并提供了关于员工参与度和创新行为的一些具体数据。"
